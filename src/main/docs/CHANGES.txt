  - Profiled search and found that the compressed stored fields made search awfully slow. Decompression
    takes up to 60% search time.

    Results from a test:
      # of images in an index of OpponentHistogram = 50000
      init time of cached search: ms = 1414
      100 searches with the cached searcher: ms = 4186
      100 searches with the non-cached version: ms = 106879
